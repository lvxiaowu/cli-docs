## 快速开发

快速创建一个新工程

```bash

// 全局安装 @lv/cli
$ npm i @lvxiaowu/cli -g

// 更新
$ lv update

// 初始化工程
$ lv init

// 创建工程
$ lv create project-name

// 启动工程
$ cd project-name && npm start
```
